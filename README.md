# Hand Gravity

Modern webcam-based hand visualizer using MediaPipe Hands, React, TypeScript, Vite and TailwindCSS.

Quick start:

1. npm install
2. npm run dev

This project tracks up to 2 hands, draws neon landmarks, energy connections, simple particles, and recognizes gestures.

Notes:
- Camera permission is requested automatically.
- Mobile browsers supported, but performance varies by device.
